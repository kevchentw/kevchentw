在Ubuntu 18.04 安裝讀卡機和健保卡元件

Posted on Sat, May 2, 2020 taiwan life
又到了五月報稅季，第一次來挑戰在 Ubuntu 上報稅

準備

  1. 讀卡機一台 EZ100PU
  2. 擁有 Root 權限的 Ubuntu

開始安裝

1. 安裝 PC/SC Smart Card 套件

sudo apt install pcscd pcsc-tools

2. 安裝 EZ100PU 驅動

sudo -s
wget http://www.castlestech.com/wp-content/uploads/2016/08/201511920271676073.zip
unzip 201511920271676073.zip
cd 201511920271676073/EZUSB_Linux/EZUSB_Linux_x86_64_v1.5.3/driver_ezusb_v1.5.3_for_64_bit
./install

3. 安裝 健保卡驅動程式

sudo -s
wget https://cloudicweb.nhi.gov.tw/cloudic/system/SMC/mLNHIICC_Setup.Ubuntu.zip
unzip mLNHIICC_Setup.Ubuntu.zip
cd mLNHIICC_Setup
./Install

4. 重新開機

5. 插上讀卡機及健保卡

6. 解決憑證錯誤

建議使用 Chrome  (用 Firefox 新增例外比較麻煩)

進入 https://localhost:7777/ 會看到這個畫面

點選 Advanced → Proceed to localhost

7. 進入所得稅網站 → 點選「健保卡＋註冊密碼」→ 看到此畫面就成功啦！

後記

新的報稅系統算是滿方便的，但希望在安裝驅動程式的部份官方可以寫完整清楚一點的文件，最後是到別人網站才搞定。

有趣的是在報稅網站發現一個新的認證方式 TW FidO，但需要自然人憑證才能註冊，未來可以來測試看看。希望可以解決身份認證的複雜度。
